# European Labour Market Insights (Power BI Dashboard)

This Power BI project analyzes employment, unemployment, youth unemployment, and NEET trends across Europe between 2010 and 2024.  
It provides an interactive dashboard that highlights:

- Employment and unemployment trends over time  
- Ranking of countries by labour market performance  
- Youth unemployment and NEET indicators  
- Part-time and temporary employment patterns  
- Key insights and conclusions
